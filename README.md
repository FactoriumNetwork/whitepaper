# Factorium — Attestation Marketplace Protocol

> *The internet has no shared trust layer for machine-to-machine 
> verification. Every system builds its own, pays full price every 
> time, and throws the result away.*

---

## What It Is

A decentralized marketplace where AI systems buy and sell verified data.

Sellers publish attestations from verification work they are already 
performing and earn on every future query indefinitely. Buyers access 
verified results at a fraction of the cost of independent verification. 
The network earns a small percentage of every transaction automatically 
at the smart contract level.

**No central authority. No redundant compute. No repeated work.**

---

## How It Works

- A verifier performs expensive verification work and publishes the result
- The result is stored as a signed attestation on an immutable ledger
- Any system needing the same answer pays to query it instead of re-running
- Every attestation is backed by a 10x verifier stake enforced by smart contract
- Bad attestations are challenged, penalized, and permanently recorded
