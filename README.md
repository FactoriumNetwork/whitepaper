# Factorium

A decentralized attestation marketplace where AI systems buy and sell verified data.

Sellers monetize expensive verification compute they are already performing.
Buyers access verified results at a fraction of the cost of independent verification.
The network earns on every transaction automatically at the smart contract level.

No central authority. No redundant compute. No repeated work.
